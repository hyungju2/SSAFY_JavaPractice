Ajax에서 사용되는 데이터 형식
1) JSON
-> 이름/값으로 이루어진 데이터의 집합.
중괄호 내에 key(문자열),value(JSON 데이터)로 구성

2) JSON 데이터의 활용
eval()함수 -> JSON 형식의 문자열 데이터를 자바스크립트 객체로 변환. 

객체를 하나 만들어서 사용할 때, '.'으로 접근한다면 그 자체가 키가 됨.
배열로 사용하는 것이 좋음.
=> user 객체에 접근하는 경우.
ex) for(var key in user){
	document.write(user[key]);
}

3) Ajax 통신
폼 전송 방식이 아닌 자바스크립트 객체를 이용한 통신.
비 동기 요청을 웹이 받으면, 콜백 함수를 실행한다. (화면이 Refresh되지 않음)
비동기는 응답이 돌아오지 않아도 다음 동작을 수행할 수 있다.

폼테그의 단점은 화면이 Refresh가 된다. 
Ajax는 화면의 전환 없이 요청한 자료를 전송받을 수 있으며, 기다릴 필요 없이 다른 작업을 바로 수행할 수 있다.
